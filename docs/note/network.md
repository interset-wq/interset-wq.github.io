## 一、网络系统结构与设计的基本原则

### 1.1 计算机网络的基本结构

典型的计算机网络从逻辑功能上分为 **资源子网** 和 **通信子网** 两部分。

- **资源子网** 负责全网的数据处理业务，负责向网络用户提供各种网络资源与网络服务。资源子网由主计算机系统、各种终端与联网设备、各种软件与信息资源等组成。
- **通信子网** 负责进行网络数据传输、转发等通信处理任务，由通信线路、通信控制处理机与其他通信设备组成。通信控制处理机是通信子网中的主要设备，也被称为网络节点。通信线路为通信控制处理机之间、通信控制处理机与主机之间提供通信通道.目前使用较为普遍的通信线路有双绞线,同轴电缆,光纤等有机介质,以及微波,卫星等无线通信信道.

### 1.2 计算机网络的分类及其互联方式

- 局域网(Local Area Network, LAN)

    指在较小地理范围内（如一栋建筑物、一个校园或一个企业）连接的计算机网络，覆盖范围通常在几米到几千米内.

    ???+ info "核心特点"
        - 覆盖范围小：一般局限于家庭、办公室、校园等局部区域。
        - 传输速度快：常用以太网技术，传输速率可达 100Mbps 至 10Gbps，甚至更高（如万兆局域网）。
        - 低延迟：设备间距离近，数据传输延迟低，适合实时通信（如视频会议、在线游戏）。
        - 组网灵活：可通过交换机、路由器等设备快速搭建，支持有线（网线）或无线（Wi-Fi）连接。
        - 管理方便：由单一机构（如家庭、企业）自主管理，安全性和隐私性较易控制。
        
        典型应用: 家庭网络（连接手机、电脑、智能设备）,企业办公网络（文件共享、打印机共享）,    校园网（教学楼、图书馆的网络互联）

- 城域网(Metropolitan Area Network, MAN)

    覆盖一个城市或地区（范围约 10-100 公里）的网络，用于连接同一城市内的多个局域网或机构网络。

    ???+ info "核心特点"
        - 覆盖范围中等：介于局域网和广域网之间，通常覆盖一个城市
        - 高速传输：采用光纤等高速传输介质，传输速率可达 100Mbps 至 10Gbps，支持大量用户同时接入。
        - 多机构互联：主要用于连接城市内的政府部门、企业、医院、学校等局域网，实现区域内的数据共享和通信。
        - 运营商主导：一般由电信运营商或政府机构建设和管理，属于公共基础设施。
        
        典型应用: 城市公交 WiFi 网络、智能交通管理系统, 城市政务网（连接各政府部门的信息系统）,  大型企业的城市分部网络互联。

- 广域网(Wide Area Network,WAN)

    覆盖范围最广的网络，可跨越国家、大洲，甚至全球（如互联网），通过多种通信链路（如光纤、卫星、微波）连接不同地区的网络

    ???+ info "核心特点"
        - 覆盖范围广：从数百公里到全球，连接不同城市、国家或大陆。
        - 传输速度差异大：受距离和链路类型影响，速度可能从低速（如拨号上网）到高速（如海底光纤）不等，但整体延迟较高。
        - 复杂的网络架构：由多个路由器、交换机和通信链路组成，需通过 “路由协议” 实现跨区域数据传输（如互联网使用 TCP/IP 协议）。
        - 多运营商合作：通常由多个网络服务提供商（ISP）共同构建和维护，属于全球性公共网络。
        
        典型应用: 互联网（Internet）：连接全球各地的计算机和网络，实现信息共享和通信,  跨国企业的全球分支机构网络（如跨国公司的总部与海外分公司互联）,国际通信网络（如海底光缆连接各大洲，实现跨国数据传输）。

???+ note "计算机网络的互联方式"
    广域网,城际网,局域网是构成现代网络系统的基本单元.用户计算机可以通过局域网、有线电视网（CATV）、电话交换网（PSTN）、无线城域网冈（WMAN）或无线局域网（WLAN）等方式接入到作为地区级主干网的城域网中。城域网又通过路由器与光纤接入到作为国家级或区域主干网的广域网中。
